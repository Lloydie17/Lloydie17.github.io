About this activity

This is my first task as barangay volunteer to create a official website of our barangay. The goal of this website is to record all activities, services, brgy. history, brgy. profile, and brgy. officials. This task was created during the pandemic for the residents who don't want to visit the barangay to request for barangay clearance or permit.

I created this website using weebly, it is a free website builder for those people who don't know how to code. The tools are already provided but I was the one who design the website. This website have 40+ visitors and also a total of 15 request form.

I was a Grade 12 student when I created this website and also with this experience it is the reason why I pursue the BSIT degree.
