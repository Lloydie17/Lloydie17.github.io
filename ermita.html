About this activity

This is the project for the barangay, we created this to store all the data or personal information of the residents in our barangay. The purpose of this project is to determine if how many residents and household are there in our barangay. Also incase of emergency such as: earthquake, fire, flood and etc. through this project we will know if what areas or families are affected and how many casualties.

This project is so very advance for me as a second year college student but because of this I really study in advance. The language we use is PHP and also MySQL is to manage databases and servers. Before this project was created me and my cousin propose this project to our barangay captain, luckily our barangay captain wanted to take advantage of technology and according to him that our barangay was the first barangay in Cebu City to have our own Database. The outcome of this project is effective it is useful for the future of our barangay.

In my experience I can now understand why programmer or a Web Developer is not an easy job, that in order to have this title you really must have the skills and also HTML and CSS are not the only tool that a Web Developer use.

My contribution to this project is that I'm the one who planned the design, ideas, information and etc. also I did some of the front-end works.
